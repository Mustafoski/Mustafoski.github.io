# Box Blur

## The Box Blur algorithm is a simple image processing technique used to blur an image by taking the average color value of a pixel and its neighbors. It is the foundation for more advanced blurs like 



---

```
const boxBlur = function (image, number) {
  const res = [];

  for(let y = 0; y < image.length - 2; y++){
    const line = [];

    for (let x = 0; x < image[y].length - 2; x++){
      let sum = 0;
      let count  = 0;


      for (let a = y; a<y +3 ;a++){
        for (let b = x; b < x+3;b++ ){
          sum += image[a][b];
          count ++
        }
      }
      line.push(Math.floor(sum/count))
    }
    res.push(line)
  }
  return res
}


```
